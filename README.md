# 채팅 서비스 고도화 (Chat Service Enhancement)

## 프로젝트 개요
이 프로젝트는 **YEAHPEU** 프로젝트의 채팅 기능을 기반으로, 성능과 확장성을 극대화하여 새롭게 구축하는 고도화 프로젝트입니다. 기존 시스템의 한계를 분석하고, 데이터베이스 설계부터 애플리케이션 구현, 성능 튜닝에 이르기까지 전 과정을 심도 있게 다룹니다.

## 주요 목표
다음과 같은 핵심 마일스톤을 단계별로 진행합니다.

1.  **기존 시스템 성능 분석**: 기존 채팅 서비스의 성능을 측정하고 부하 테스트를 수행하여 현재의 병목 구간과 개선 필요 사항을 명확히 파악합니다.
2.  **데이터베이스 재설계**: 대용량 트래픽과 데이터를 효율적으로 처리할 수 있는 최적의 데이터베이스 스키마를 백지상태에서 다시 설계합니다.
3.  **서비스 재구현**: 클린 코드 원칙과 성능을 고려하여 채팅 서비스 로직을 새롭게 구현합니다.
4.  **신규 시스템 성능 검증**: 새롭게 구축된 시스템에 대해 강도 높은 부하 테스트를 수행하여 안정성과 성능 목표 달성 여부를 검증합니다.
5.  **극한의 성능 튜닝**: 데이터베이스 쿼리 튜닝, 캐싱 전략, 비동기 처리 등 전체 스택에 걸친 최적화를 통해 최대 처리량과 초저지연을 달성합니다.
